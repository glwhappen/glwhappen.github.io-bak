  • 1.2.2 - 认识各个硬件部件
    • 这节课还是比较硬核的
    • 字(word): 跟计算机的硬件有关 如果是32位系统则一个字是4个字节,如果是64位,则是8个字节。
    • 字节(Byte): 1字节 = 8bit
    • 运算器
      • ACC：累加器 Accumulator
      • MQ：乘商寄存器 Multiple-Quotient Register
      • X：通用的操作数寄存器
      • ALU：算术逻辑单元 Arithmetic and Logic Unit
    • 控制器
      • CU：控制单元 Control Unit
      • IR：指令寄存器 Instruction Register
      • PC：程序寄存器 Program Counter
