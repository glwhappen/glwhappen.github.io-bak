如何发布自己的npm包

本文最后更新于：2022年2月22日 晚上

经常用npm安装各种包，现在来试试自己发布一个吧。

1. 注册账号

https://www.npmjs.com/

2. 发送验证邮件

一定要记得验证，要不会一直报错

3. 切换镜像站

一定要切换到官方镜像站

4. 登录

这里一定要切换到官方镜像才能登录成功

5. 创建目录

可以npm init一个目录，或者直接GitHub克隆一个。
然后再package.json文件中修改以下信息

package name：比较重要，会显示到npm仓库，不能和现有的名称重复
version: 版本号
license：开源协议

5. 发布

进入包所在的目录

第一个包发布成功
